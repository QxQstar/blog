# 关于我

工作这些年，我越来越强烈地感到，很多人的问题不是努力不够，能力太差，更常见的情况是，看不清自己所处的系统，不知道规则怎样运转，不知道资源掌握在谁手里，也不知道别人的真实激励是什么。于是，投入了很多时间和精力，最后得到的回报却很有限。

我写这个博客是想持续做一件事：把阅读、观察、工作经验和现实案例整理成可复用的判断框架。

这个博客主要写四类内容：

## 职场决策

写求职、面试、薪酬、晋升、离职与管理。

关注个体如何在工作中做更划算的选择。

## 组织观察

写权力、协作、资源分配、话语权与办公室政治。

关注组织表面流程背后的真实激励。

## 关系社会

写亲密关系、性别、家庭与社会结构。

关注私人困惑背后的制度压力与现实约束。

## 现实案例

拆平台、公司、职业路径与具体事件。

通过现实样本训练判断力，而不是停留在观点表态。

题材很多，最后都会落在：

1. 人在系统里怎么被影响。  
2. 规则背后的真实激励是什么。  
3. 怎样把手里的资源，慢慢变成自己的选择权。

这个博客面向想提升判断力的工薪阶层，如果你也关心这些问题，也想少一点误判，少吃一点信息差的亏，慢慢积累自己的资产和选择权，那么这里写的内容，可能对你有用。

我不保证提供标准答案，但会尽量把看到的、想明白的、做过验证的东西写清楚，你可以把这里当成一个现实分析样本库。
